# androidapn/

Module principal du projet AndroidAPN.

## Contenu
- `core.py` : logique centrale pour la génération et la validation des configurations APN.
- `__init__.py` : initialise le module Python.

## Rôle
Ce dossier contient le code source du projet :
- traitement des données APN,
- génération des fichiers `apn.conf`,
- intégration avec les scripts CLI.

## Notes
Le code est conçu pour être modulaire, testable et extensible.
