# docs/

Documentation du projet AndroidAPN.

## Contenu
- `index.rst` : page d’accueil de la documentation.
- `introduction.rst` : présentation du projet.
- `usage.rst` : guide d’utilisation.
- `apn_format.rst` : structure d’un fichier APN.
- `architecture.rst` : organisation du projet.

## Objectif
Fournir une base claire et concise pour comprendre, utiliser et étendre AndroidAPN.

## Notes
Les fichiers sont au format reStructuredText (RST) et compatibles avec Sphinx.
