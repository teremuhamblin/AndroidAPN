======
Usage
======

Générer un fichier APN
----------------------

.. code-block:: bash

   python scripts/generate_apn_db.py --input source.yaml --output config/apn.conf

Pré-requis
----------

- Python 3.10+
- Installation des dépendances :

.. code-block:: bash

   pip install -r requirements.txt

Notes
-----

Les logs sont générés dans ``logs/androidapn.log``.
