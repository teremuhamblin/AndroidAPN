AndroidAPN Documentation
========================

AndroidAPN est un projet open-source permettant de lire, exporter,
valider et générer des configurations APN pour appareils Android.

Cette documentation couvre :

- Architecture du projet
- Format APN (YAML → Android)
- Base de données APN FR/EU
- Utilisation ADB pour extraction
- CLI Python (optionnelle)
- Application Android (optionnelle)
