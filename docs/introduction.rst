Introduction
============

AndroidAPN est un outil conçu pour standardiser et simplifier la gestion
des APN (Access Point Names) sur Android.

Objectifs du projet
-------------------

- Fournir un format APN lisible (YAML)
- Générer automatiquement des fichiers APN compatibles Android
- Exporter les APN d’un appareil via ADB
- Documenter les APN FR/EU
- Préparer une future application Android

Public visé
-----------

- Techniciens télécom
- Administrateurs systèmes
- Développeurs Android
- Utilisateurs avancés
