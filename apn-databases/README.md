# APN France — Base officielle AndroidAPN

Ce dossier contient les APN officiels des opérateurs français, convertis
au format YAML standardisé AndroidAPN.

Sources officielles :
- Orange France
- SFR (Assistance SFR)
- Bouygues Telecom (Assistance Bouygues)
- Free Mobile (Assistance Free)

MVNO inclus :
- NRJ Mobile
- CIC Mobile
- Auchan Telecom
- La Poste Mobile
- Prixtel

Format utilisé : YAML normalisé AndroidAPN.
