# logs/

Ce dossier contient les fichiers de logs générés par AndroidAPN.

## Contenu
- Les logs sont créés automatiquement par le logger `androidapn`.
- Le fichier principal est `androidapn.log` (rotation possible selon configuration).

## Notes
- Les fichiers `.log` ne sont pas versionnés (voir `.gitignore`).
- Le fichier `.gitkeep` permet de conserver le dossier dans le dépôt.
